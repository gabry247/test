Modalità di Trasmissione
----------------------------

Dati di dettaglio della SAP e della SEZIONE 6 – POLITICHE ATTIVE saranno inviati tramite i servizi informatici messi a disposizione dai servizi competenti, 
secondo le modalità definite. L’invio del flusso SAP e SEZIONE 6 – POLITICHE ATTIVE avverrà mediante l’invio di un file in formato XML. Il file XML contiene una sola scheda SAP.
